# EU–China Innovation Tracker

[![Live Dashboard](https://img.shields.io/badge/Dashboard-Live-4fc3f7)](https://tarrrrra.github.io/eu-china-innovation-tracker/eu_china_sti_tracker_dashboard.html)
[![Tableau Public](https://img.shields.io/badge/Tableau-Public-ff9f43)](https://public.tableau.com/views/EU-ChinaInnovationTracker/Sheet1)

Automated policy tracker for the EU–China Science & Technology Cooperation Agreement (2000–Present).
Scrapes Xinhua (EN/ZH), People's Daily, and EU Commission sources with NLP classification across 6 activity types.
